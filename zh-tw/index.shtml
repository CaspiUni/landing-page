openSUSE

openSUSE 是一個自由的、基於 Linux 的作業系統， 專為您的個人電腦、筆記型電腦或伺服器所設計。 您可以暢遊網路、管理您的電子郵件與照片、 執行辦公室工作、欣賞影片與音樂。 have a lot of fun!

  •   Get it: 下載 openSUSE 或尋找其他軟體  
  •   Discover it: 使用教學、文件以及其他資源了解更多關於 openSUSE 的事！  
  •   Create it: 使用 BuildService 為所有主要 Linux 發行版本建構套件。
