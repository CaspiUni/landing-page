openSUSE

openSUSE はPC、ラップトップ、 サーバ用のフリーな、 Linux をベースとしたオペレーティングシステムです。 Web の閲覧、e-mail や写真の管理、 オフィスでの仕事、 動画や音楽の再生といった用途に使え、 あなたに多くの楽しみをもたらすでしょう。 have a lot of fun!

  •   Get it: openSUSE をダウンロード / 追加ソフトウエアを探す。  
  •   Discover it: チュートリアル、文書、その他多くの資料で openSUSE についてもっと知る！  
  •   Create it: ビルドサービスを使ってあらゆるメジャーな Linux ディストリビューション向けパッケージを構築する。
