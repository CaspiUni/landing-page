openSUSE

openSUSE 是一个自由的、基于 Linux 的操作系统，适用于您的个人电脑、笔记本电脑或服务器。您可以畅游网络、管理您的电子邮件与照片、办公、欣赏视频与音乐。 have a lot of fun!

  •   Get it: 下载 openSUSE 或寻找其他软件  
  •   Discover it: 使用教程、文档以及其他资源了解更多关于 openSUSE 的事！  
  •   Create it: 使用 BuildService 为所有主要 Linux 发行版构建软件。
